Phase 1 系统架构

最小可用版本的架构设计

整体架构
👑 管理员
创建/编辑 SOP
👥 同事
选择 SOP + 上传数据
↓          ↓
🌐 Web 界面 (Vue)
SOP 管理  |  文件上传  |  执行结果
↓
⚙️ 后端服务 (FastAPI)
SOP 解析器
代码生成器
沙箱执行器
↓
📁 文件系统
Excel / CSV 输入输出
💾 SQLite
SOP 存储  |  执行记录

技术选型

前端 Vue 3 + Vite + Tailwind 现有项目复用
后端 Python FastAPI 轻量易部署
数据库 SQLite 本地文件，无需安装
代码执行 subprocess 沙箱 Phase 1 简化版
