## 1. 业务目标与范围

### 1.1 核心问题

教练每日上课结束后上传课堂照片，但现有签到系统中的 `actual_count`（签到人数）仅来自教练手工填写，存在以下漏洞：

- 误记、少记、多记
- 照片中学员与签到名单对不上

### 1.2 解决方案

引入**视觉 AI 辅助核对**：自动识别照片中的人数，与 `daily_checkin` 表中的签到人数对比，差值异常时标红提示，供管理员人工复核。

### 1.3 三方校验模型

```
daily_checkin 表 (已有)
  ├── expected_count  → 课程应到人数（排课系统）
  ├── actual_count    → 签到系统实际数（教练填写）
  └── (新增关联)
        ↓
photo_checkin_records 表 (新建)
  └── detected_count  → AI 识别照片中的人数
        ↓
  差值 = detected_count - actual_count
  超出阈值 → 高亮标记 → 人工复核
```

### 1.4 范围边界

**本次实现包含**：
- YOLO 本地推理 API
- 照片与签到记录手动关联
- 单次/批量照片上传
- 识别结果展示（标注图 + 人数）
- 差值核对汇总表
- 人工修正入口

**本次不包含（留后续迭代）**：
- EXIF 时间自动匹配课程（照片时间 → 自动找最近签到记录）
- 学员人脸识别（隐私合规问题需额外评估）
- 历史照片批量补跑

## 2. 技术选型决策

### 2.1 人体检测模型：YOLOv8n

**选择理由**：

| 维度 | YOLOv8n | Claude Vision | OpenCV HOG |
|---|---|---|---|
| 运行方式 | 完全本地 | 需联网+计费 | 完全本地 |
| 精度（室内人群） | ★★★★ | ★★★★★ | ★★ |
| 速度（CPU） | 1-3s/张 | 2-5s/张+网络延迟 | 0.5s/张 |
| 部署复杂度 | 低（pip install） | 低（API调用） | 低 |
| 适合本项目 | ✅ 首选 | 备用（YOLO精度不足时） | ✗ |

**模型文件**：`yolov8n.pt`，约 6MB，首次运行自动下载到 `~/.cache/ultralytics/`

**只检测 person 类**：COCO 数据集 class 0 = person，其余 79 类全部忽略。

### 2.2 误入人员过滤策略

体育教学场景特征：学员聚集在画面中央（球场/教室），路人偏画面边缘且距离较远（bbox 较小）。

采用组合过滤，按序应用：

```
原始检测框列表（class=person）
  ↓ [过滤1] 置信度阈值：去掉 confidence < 0.45 的框
  ↓ [过滤2] 最小面积过滤：bbox 面积 / 图片总面积 < 0.008（0.8%）的极小目标去掉
  ↓ [过滤3] 边缘区域过滤：bbox 中心点落在图片上下各 5% 的极边缘区域的框降权
             （默认仍计入，但标记为 low_confidence）
  ↓ 最终有效人数
```

**可配置参数**（存入 `photo_checkin_records.filter_config` JSON 字段，方便后续调参）：

```json
{
  "conf_threshold": 0.45,
  "min_bbox_ratio": 0.008,
  "edge_margin_pct": 0.05
}
```

### 2.3 标注图返回方案

推理后用 PIL 在原图上绘制检测框，返回 Base64 编码的 JPEG，前端直接 `<img :src="'data:image/jpeg;base64,...'" />` 展示，无需额外文件服务。

## 3. 系统架构设计

### 3.1 整体流程

```
前端 PhotoCheckin.vue
  │
  ├─[Step 1] 选择日期 → GET /api/tools/photo-checkin/sessions?date=YYYY-MM-DD
  │           → 返回当日签到课程列表（供关联选择）
  │
  ├─[Step 2] 选择课程 + 上传照片（支持多图）
  │           → POST /api/tools/photo-checkin/detect
  │              body: checkin_id, photos[]
  │           → 返回每张图的 { detected_count, annotated_image_b64, boxes[] }
  │
  ├─[Step 3] 展示识别结果
  │           → 标注图（可切换原图/标注图）
  │           → 检测人数 vs 签到人数 差值
  │           → 人工修正输入框
  │
  └─[Step 4] 确认保存
              → POST /api/tools/photo-checkin/confirm
                 body: checkin_id, records[]
              → 写入 photo_checkin_records 表
```

### 3.2 文件存储

照片原图**不持久化存储**到磁盘（避免存储膨胀），仅在内存中完成推理后丢弃。  
标注图也仅通过 API 响应返回，不落盘。

> **例外**：如果后续需要审计追溯，可在 `data/photo_uploads/{YYYY-MM-DD}/` 按需保存，但本次规划不实现。

        UPDATE photo_checkin_records

        WHERE id = ?
        """,

    )
    conn.commit()
    conn.close()

def get_photo_records_by_checkin(checkin_id: int) -> List[Dict[str, Any]]:
    """查询某条签到记录下的所有照片检测结果。"""

    rows = conn.execute(
        "SELECT * FROM photo_checkin_records WHERE checkin_id = ? ORDER BY created_at",

               d.actual_count, d.expected_count

        LEFT JOIN daily_checkin d ON p.checkin_id = d.id

        ORDER BY p.created_at
        """,
        (check_date, check_date),
    ).fetchall()
    conn.close()
    return [dict(r) for r in rows]

## 5. 后端实现规范

### 5.1 新建文件：`backend/tools/photo_checkin.py`

**完整实现模板**（按此结构编写）：

```python
"""
photo_checkin.py
照片人数识别核心模块：YOLO 推理 + 过滤 + 标注图生成
"""
import io
import json
import base64
import logging
from typing import List, Dict, Any, Optional, Tuple

logger = logging.getLogger(__name__)

# ──────────────────── 默认过滤参数 ────────────────────

DEFAULT_FILTER_CONFIG = {
    "conf_threshold": 0.45,      # 置信度阈值
    "min_bbox_ratio": 0.008,     # bbox 最小面积占比（相对图片总面积）
    "edge_margin_pct": 0.05,     # 边缘降权区域（上下各 5%）
}

# ──────────────────── 模型懒加载 ────────────────────
# 用模块级变量缓存模型，避免每次请求重新加载（~2-3s 开销）

_model = None

def _get_model():
    global _model
    if _model is None:
        try:
            from ultralytics import YOLO
            _model = YOLO("yolov8n.pt")   # 首次运行自动下载到 ~/.cache/ultralytics/
            logger.info("YOLOv8n model loaded successfully")
        except ImportError:
            raise RuntimeError(
                "ultralytics 未安装，请运行: pip install ultralytics"
            )
    return _model


# ──────────────────── 核心推理函数 ────────────────────

def detect_persons(
    image_bytes: bytes,
    filename: str,
    filter_config: Optional[Dict] = None,
) -> Dict[str, Any]:
    """
    对单张图片进行人体检测。

    Returns:
        {
            "detected_count": int,       # 过滤后有效人数
            "raw_count": int,            # 过滤前原始检测框数
            "confidence_avg": float,     # 有效框平均置信度
            "annotated_image_b64": str,  # Base64 编码的标注图 JPEG
            "boxes": [...],              # 每个检测框详情（供前端可选展示）
            "filter_config": dict,       # 实际使用的过滤参数
        }
    """
    from PIL import Image, ImageDraw, ImageFont
    import numpy as np

    cfg = {**DEFAULT_FILTER_CONFIG, **(filter_config or {})}

    # 1. 解码图片
    img = Image.open(io.BytesIO(image_bytes)).convert("RGB")
    img_w, img_h = img.size
    img_area = img_w * img_h

    # 2. YOLO 推理（仅 person 类）
    model = _get_model()
    results = model(img, classes=[0], verbose=False)  # class 0 = person
    detections = results[0].boxes  # Boxes object

    # 3. 解析检测框
    raw_boxes = []
    if detections is not None and len(detections) > 0:
        for box in detections:
            x1, y1, x2, y2 = box.xyxy[0].tolist()
            conf = float(box.conf[0])
            raw_boxes.append({"x1": x1, "y1": y1, "x2": x2, "y2": y2, "conf": conf})

    raw_count = len(raw_boxes)

    # 4. 应用过滤规则
    valid_boxes = []
    for b in raw_boxes:
        # 过滤1：置信度
        if b["conf"] < cfg["conf_threshold"]:
            continue
        # 过滤2：最小面积
        bbox_area = (b["x2"] - b["x1"]) * (b["y2"] - b["y1"])
        if bbox_area / img_area < cfg["min_bbox_ratio"]:
            continue
        # 过滤3：极边缘区域（仅标记，不直接过滤）
        cx = (b["x1"] + b["x2"]) / 2
        cy = (b["y1"] + b["y2"]) / 2
        edge_y = img_h * cfg["edge_margin_pct"]
        b["edge_flag"] = cy < edge_y or cy > (img_h - edge_y)
        valid_boxes.append(b)

    detected_count = len(valid_boxes)
    confidence_avg = (
        sum(b["conf"] for b in valid_boxes) / detected_count
        if detected_count > 0 else 0.0
    )

    # 5. 生成标注图
    annotated_img = img.copy()
    draw = ImageDraw.Draw(annotated_img)

    for b in valid_boxes:
        color = "#FF6B35" if not b.get("edge_flag") else "#FFC300"  # 橙色/黄色区分边缘
        draw.rectangle([b["x1"], b["y1"], b["x2"], b["y2"]], outline=color, width=3)

    # 在左上角叠加计数文字
    draw.rectangle([0, 0, 160, 40], fill=(0, 0, 0, 180))
    draw.text((10, 10), f"识别人数: {detected_count}", fill="white")

    # 转 Base64
    buf = io.BytesIO()
    annotated_img.save(buf, format="JPEG", quality=85)
    b64 = base64.b64encode(buf.getvalue()).decode("utf-8")

    return {
        "detected_count": detected_count,
        "raw_count": raw_count,
        "confidence_avg": round(confidence_avg, 3),
        "annotated_image_b64": b64,
        "boxes": [
            {"conf": round(b["conf"], 3), "edge_flag": b.get("edge_flag", False)}
            for b in valid_boxes
        ],
        "filter_config": cfg,
    }


def detect_multiple(
    images: List[Tuple[str, bytes]],   # [(filename, bytes), ...]
    filter_config: Optional[Dict] = None,
) -> List[Dict[str, Any]]:
    """批量处理多张图片，返回列表。"""
    results = []
    for filename, image_bytes in images:
        try:
            result = detect_persons(image_bytes, filename, filter_config)
            result["filename"] = filename
            result["error"] = None
        except Exception as e:
            logger.error(f"detect_persons failed for {filename}: {e}")
            result = {
                "filename": filename,
                "detected_count": 0,
                "raw_count": 0,
                "confidence_avg": 0.0,
                "annotated_image_b64": None,
                "boxes": [],
                "filter_config": filter_config or DEFAULT_FILTER_CONFIG,
                "error": str(e),
            }
        results.append(result)
    return results
```

### 5.2 新建路由段：`backend/api/photo_checkin_routes.py`

```python
"""
photo_checkin_routes.py
照片人数核对 API 路由（独立文件，不影响 tool_routes.py）
"""
from fastapi import APIRouter, HTTPException, UploadFile, File, Form, Query
from fastapi.responses import JSONResponse
from typing import Optional, List
import json

from tools.photo_checkin import detect_multiple, DEFAULT_FILTER_CONFIG
from tools.db import (
    get_sessions_by_date,
    save_photo_record,
    update_photo_record,
    get_photo_records_by_checkin,
    get_photo_records_by_date,
)

router = APIRouter()

ALLOWED_EXTENSIONS = {".jpg", ".jpeg", ".png", ".webp", ".heic"}
MAX_FILE_SIZE_MB = 20


# ──────────────────── 查询当日课程（关联选择用） ────────────────────

@router.get("/tools/photo-checkin/sessions")
async def get_sessions(date: str = Query(..., description="YYYY-MM-DD")):
    """返回指定日期的签到课程列表。"""
    try:
        sessions = get_sessions_by_date(date)
        return {"date": date, "sessions": sessions, "count": len(sessions)}
    except Exception as e:
        raise HTTPException(status_code=500, detail=str(e))


# ──────────────────── 上传照片 + 推理 ────────────────────

@router.post("/tools/photo-checkin/detect")
async def detect_photos(
    photos: List[UploadFile] = File(...),
    checkin_id: Optional[int] = Form(None),
    filter_config_json: Optional[str] = Form(None),
):
    """
    接收照片列表，执行 YOLO 推理，返回识别结果（不写库，仅返回）。
    前端预览确认后再调用 /confirm 写库。
    """
    if not photos:
        raise HTTPException(status_code=400, detail="未上传任何照片")
    if len(photos) > 10:
        raise HTTPException(status_code=400, detail="单次最多上传 10 张照片")

    # 解析自定义过滤参数
    filter_config = DEFAULT_FILTER_CONFIG.copy()
    if filter_config_json:
        try:
            filter_config.update(json.loads(filter_config_json))
        except json.JSONDecodeError:
            raise HTTPException(status_code=400, detail="filter_config_json 格式错误")

    # 读取所有图片
    images = []
    for photo in photos:
        ext = "." + (photo.filename or "").rsplit(".", 1)[-1].lower()
        if ext not in ALLOWED_EXTENSIONS:
            raise HTTPException(
                status_code=400,
                detail=f"不支持的图片格式 {ext}，仅支持 {ALLOWED_EXTENSIONS}"
            )
        content = await photo.read()
        if len(content) > MAX_FILE_SIZE_MB * 1024 * 1024:
            raise HTTPException(status_code=400, detail=f"图片 {photo.filename} 超过 {MAX_FILE_SIZE_MB}MB 限制")
        images.append((photo.filename or f"photo_{len(images)}.jpg", content))

    # 批量推理
    try:
        results = detect_multiple(images, filter_config)
    except RuntimeError as e:
        raise HTTPException(status_code=503, detail=str(e))

    return {
        "checkin_id": checkin_id,
        "photo_count": len(results),
        "results": results,
    }


# ──────────────────── 确认保存 ────────────────────

@router.post("/tools/photo-checkin/confirm")
async def confirm_records(body: dict):
    """
    将推理结果写入 photo_checkin_records 表。
    body: {
      "checkin_id": int | null,
      "records": [
        {
          "filename": str,
          "detected_count": int,
          "raw_count": int,
          "confidence_avg": float,
          "filter_config": dict
        }, ...
      ]
    }
    """
    checkin_id = body.get("checkin_id")
    records = body.get("records", [])
    if not records:
        raise HTTPException(status_code=400, detail="records 不能为空")

    saved_ids = []
    for rec in records:
        new_id = save_photo_record(
            checkin_id=checkin_id,
            photo_filename=rec.get("filename", "unknown"),
            detected_count=rec.get("detected_count", 0),
            raw_count=rec.get("raw_count", 0),
            confidence_avg=rec.get("confidence_avg", 0.0),
            filter_config=rec.get("filter_config", DEFAULT_FILTER_CONFIG),
            status="confirmed",
        )
        saved_ids.append(new_id)

    return {"saved": len(saved_ids), "ids": saved_ids}


# ──────────────────── 人工修正 ────────────────────

@router.patch("/tools/photo-checkin/records/{record_id}")
async def patch_record(record_id: int, body: dict):
    """
    人工修正：提交修正人数、备注、状态。
    body: { "manual_count": int, "note": str, "status": "confirmed" | "flagged" }
    """
    manual_count = body.get("manual_count")
    note = body.get("note", "")
    status = body.get("status", "confirmed")
    if status not in ("confirmed", "flagged", "pending"):
        raise HTTPException(status_code=400, detail="status 无效")
    ok = update_photo_record(record_id, manual_count, note, status)
    return {"success": ok}


# ──────────────────── 查询历史 ────────────────────

@router.get("/tools/photo-checkin/records")
async def get_records(
    date: Optional[str] = Query(None),
    checkin_id: Optional[int] = Query(None),
):
    """查询历史检测记录（按日期或签到ID）。"""
    if checkin_id:
        records = get_photo_records_by_checkin(checkin_id)
    elif date:
        records = get_photo_records_by_date(date)
    else:
        raise HTTPException(status_code=400, detail="需要提供 date 或 checkin_id 参数")
    return {"records": records, "count": len(records)}
```

**Step 3 逻辑（差值计算 + 保存）**：

```typescript
// 计算差值（用于高亮标记）
const DIFF_THRESHOLD = 2  // 差值超过 2 人才高亮

const diffSummary = computed(() => {
  const detected = detectResults.value.reduce((sum, r) => sum + r.detected_count, 0)
  const session = sessions.value.find(s => s.id === selectedCheckinId.value)
  const actual = session?.actual_count ?? null
  return {
    detected,
    actual,
    diff: actual !== null ? detected - actual : null,
    abnormal: actual !== null ? Math.abs(detected - actual) > DIFF_THRESHOLD : false,
  }
})

async function saveResults() {
  loading.value = true
  try {
    const body = {
      checkin_id: selectedCheckinId.value,
      records: detectResults.value.map(r => ({
        filename: r.filename,
        detected_count: r.detected_count,
        raw_count: r.raw_count,
        confidence_avg: r.confidence_avg,
        filter_config: r.filter_config,
      }))
    }
    const res = await fetch(`${API}/confirm`, {
      method: 'POST',
      headers: { 'Content-Type': 'application/json' },
      body: JSON.stringify(body),
    })
    if (!res.ok) throw new Error('保存失败')
    // 保存成功后加载历史记录
    await loadHistory()
    errorMsg.value = ''
  } catch (e: any) {
    errorMsg.value = e.message
  } finally {
    loading.value = false
  }
}

async function loadHistory() {
  const params = selectedCheckinId.value
    ? `checkin_id=${selectedCheckinId.value}`
    : `date=${selectedDate.value}`
  const res = await fetch(`${API}/records?${params}`)
  const data = await res.json()
  savedRecords.value = data.records
}
```
